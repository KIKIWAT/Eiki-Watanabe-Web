Bitácora

Acá registraré el progreso de mis clases. Se contará el objetivo y el proceso que se debió seguir de manera virtual y la presencial por separado. Luego se contará mi experiencia con el proceso en cada una de estas.

Sesión 3

En la sesión 3, la clase virtual tuvo como objetivo crear la página web del portafolio. Para ello se creó una cuenta en GitHub en donde colocamos varios archivos que servían como plantilla del portafolio. Luego se descargó el programa Visual Studio el cual se sincroniza con el contenido de GitHub para poder editar el contenido de la página.

En mi caso, debido a una conexión inestable, no pude seguir eficazmente la clase por lo que recurrí a ver la grabación con la cual no tuve prisa y no tuve problemas para entender las instrucciones.

No asistí a la clase presencial debido a un malestar corporal, sin embargo, mi compañera me informó que en esa clase se vió nuevamente el tema de crear la web debido a que no quedó muy claro para muchos.

Sesión 4

En la sesión 4, la clase virtual tuvo como objetivo editar el contenido de la página web. Para ello se utilizó Visual Studio empezando por el nombre al lado del logo de la página el cual la plantilla coloca el nombre del profesor. Luego se explicó cómo cambiar el texto de los párrafos y, finalmente, reemplazar las imágenes.

En mi caso tuve problemas para entender cómo reemplazar las imágenes pero fue una duda que dejé para la siguiente clase.

En la clase presencial se volvió a explicar el tema de las imágenes además de completar el código relacionado con el apartado de contactos de la página.

Durante esta clase, tuve problemas con acceder a GitHub por una demora en el envío del código de verificación para entrar con mi cuenta. Aún después de acceder, tuve problemas con sincronizar los cambios de Visual Studio a la página por lo que decidí hacerlo en casa.

Sesión 5

En esta sesión se realizaron exposiciones grupales sobre unas problemáticas establecidas y desarrolladas a lo largo del periodo del primer promedio. Además, se aclararon varios puntos y dudas sobre el desarrollo de la web.
